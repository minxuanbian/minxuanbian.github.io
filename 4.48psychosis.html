4.48 Psychosis

This work focuses on: Dislocation between memory and reality

This work began from: A moment when memory overlapped with a dream, while sound became unnaturally compressed

The work asks: When memory collapses, do you trust memory or yourself?

Form / Method: Dislocation

Year: 2025

Location: Shanghai, China

Duration: 11.2025
